Hi 大家！我是 Evan，一名獨立開發者（Indie Hacker）。最近因為需求，我親自經歷了一次在美國開公司的完整流程，從申請到報稅都親自操作了一遍。我知道很多朋友也有類似的需求，所以今天就來分享我的經驗，希望能幫助到有需要的人！

## 為什麼要開美國公司?

其實會想開美國公司，主要是因為我在開發 SaaS 產品，需要 Stripe 這個金流工具，在台灣不能用。所以呢，開個美國公司就變成了一個不錯的選擇。

除此之外，美國公司還有幾個好處:

1. 可以合法收取美金
2. 能使用更多國際化的金融服務
3. 在某些情況下，可以享有稅務優惠

不過呢，開美國公司也不是萬能的。如果你只是想做做小生意，或者金額不大，其實不一定需要特地去開。畢竟開公司還是要花一些錢的，要評估一下值不值得。

## 美國公司要選哪種類型?

在美國開公司，最常見的就是 LLC（有限責任公司）和 Corporation（股份有限公司）這兩種。

我自己是選擇 LLC，原因很簡單 - 申請容易、維護成本低，而且稅務申報相對簡單。對於我們這種小規模的網路創業者來說，LLC 真的是再適合不過了。

如果你的公司規模比較大，或者有上市計畫，那可能就要考慮 Corporation 了。不過這部分比較複雜，建議找專業人士諮詢比較好。

## 開美國公司要花多少錢?

這可能是大家最關心的問題了。根據我的經驗，整個流程大概要花：

- 申請 LLC：約 NT$3,000
- 註冊代理費用：第一年免費，之後每年約 NT$3,000
- 美國電話號碼：每年約 NT$1,000
- 報稅相關費用：每年約 NT$3,000

總計下來，第一年大概要準備 NT$7,000 左右。之後每年的維護費用在 NT$7,000 上下。

當然啦，這只是一個粗略的估計。實際花費還是要看你選擇的服務商和自己的需求而定。

## 開美國公司的完整流程

接下來就來分享一下完整的申請流程。整個過程說複雜也不複雜，但是步驟蠻多的，建議大家耐心看完喔!

### 步驟一：選擇註冊州

美國有 50 個州，每個州的法規和稅制都不太一樣。一般來說，最受歡迎的是 New Mexico、Wyoming、Nevada 和 Delaware。

我自己是選擇 Wyoming，主要是因為：

1. 沒有州所得稅
2. 年度費用便宜，只要 $60 美金

另外，如果讓代理提交年度報告的話，不需要在成立證書上列出成員名字，可以保護隱私

### 步驟二：選擇一個可靠的註冊代理

選擇一個可靠的註冊代理非常重要，他們會幫助你處理許多繁瑣的文件工作。我選擇的是 Wyoming Registered Agent Services LLC，他們的服務質量很高，價格也非常合理。

雖然網上有很多人推薦 Northwest Registered Agent，因為他們的註冊費看似較低（$39 加州費），但從第二年開始，每年的代理費是 $125。相比之下，Wyoming Registered Agent 每年的費用僅為 $25。此外，Northwest 有聯盟計劃，推薦他們的用戶可以獲得佣金，而 Wyoming Agent 並沒有，這也解釋了為什麼網上有那麼多推薦 Northwest 的原因。

#### Wyoming Registered Agent Services LLC

Wyoming Registered Agent Services LLC 是一家提供美國公司註冊服務的代理商，專門協助海外企業家在美國成立公司。

**主要服務:**

- LLC 註冊
- 提供註冊地址
- 處理政府文件
- 協助申請 EIN (雇主識別號碼)

**優點:**

1. 價格實惠：起步價僅 $25 美元
2. 首年免費註冊代理服務
3. 提供免費運營協議
4. 流程簡單，全程在線操作

**如何使用:**

1. 前往他們的官網：[Wyoming Registered Agent Services LLC](https://wyomingregisteredagent.com/)
2. 選擇你想要的服務套餐
3. 填寫必要的公司信息
4. 付款並等待處理

### 步驟三：申請 EIN (雇主識別號碼)

EIN 就像是公司的身分證號碼，在美國開設銀行帳戶或是報稅時都會用到。你可以自己向 IRS (美國國稅局)申請，完全免費，大概 2-3 天就能拿到。

不過要注意，申請 EIN 時用的地址要跟你註冊公司時用的一樣喔!

### 步驟四：申請美國電話號碼

很多美國服務都需要美國電話號碼來驗證身份。我是用 Zadarma 這個服務，一年才 $36 美金，超划算的!

#### Zadarma 美國電話號碼服務

Zadarma 是一個提供虛擬電話號碼的在線服務，特別適合需要美國電話號碼的海外用戶。

**主要特點:**

1. 提供美國虛擬電話號碼
2. 支持接收簡訊和語音通話
3. 有專屬的手機 App，方便管理
4. 價格實惠，年費僅 $36 美元

**如何申請:**

1. 前往 [Zadarma 官網](https://zadarma.com/en/?ref=767975cfb422a13d57fbbeef3cf0569a)
2. 註冊帳號
3. 選擇 "Order a Virtual Number"
4. 選擇美國，然後挑選一個喜歡的號碼
5. 完成付款

**注意:** 申請時需要提供身份證明文件，通常使用護照即可。

**使用提醒:**

- 確保選擇 "Mobile+SMS" 類型的號碼，這樣才能接收簡訊
- 建議選擇年付方案，有折扣優惠
- 下載 Zadarma App 以便隨時接聽電話或查看簡訊

Zadarma 是一個經濟實惠的選擇，特別適合需要美國電話號碼來驗證各種服務的海外用戶。

### 步驟五：開設美國銀行帳戶

有了公司和 EIN 之後，就可以開設美國銀行帳戶了。我用的是 Mercury，這家網路銀行對海外用戶特別友善，而且完全免費!

#### Mercury 銀行開戶指南

Mercury 是一家專為創業者和小企業設計的美國網路銀行，特別適合海外用戶。

**主要特點:**

1. 完全免費的商業支票帳戶
2. 支持 ACH、電匯等多種轉帳方式
3. 提供虛擬和實體借記卡
4. 用戶友好的網頁和移動應用界面
5. 提供 API 接口，方便與其他服務集成

**開戶步驟:**

1. 訪問 [Mercury 官網](https://mercury.com/)
2. 點擊 "Open an Account" 開始申請流程
3. 填寫公司信息，包括 EIN、公司地址等
4. 上傳所需文件，如公司成立證書、護照等
5. 等待審核，通常需要 1-2 個工作日

**需要準備的文件:**

- 公司成立證書
- EIN 證明
- 護照或其他身份證明
- 地址證明（英文版）
- 公司運營協議（如果是 LLC）

**注意事項:**

- 確保提供的所有信息與公司註冊信息一致
- 地址證明需要是英文版，可以是水電費帳單或銀行對帳單
- 如果審核過程中有任何問題，Mercury 的客服會通過郵件聯繫你

Mercury 銀行的開戶流程相對簡單，而且他們的客服響應迅速，對海外用戶很友好。開戶成功後，你就可以輕鬆管理你的美國公司資金了!

開戶時需要提供英文地址證明，我是去戶政事務所申請英文戶籍謄本，大概等了兩週才拿到。

### 步驟六：申請 Stripe 帳戶

有了美國公司和銀行帳戶，就可以申請 Stripe 了!整個過程很簡單，照著網站的指示一步步來就行了。

記得要選擇美國作為你的商家所在地，然後填入你的 EIN、公司地址等資訊。

### 步驟七：提交 BOI 報告

這是 2024 年新出的規定，所有美國公司都要提交 BOI (實益所有人)報告。我是直接在 FinCEN 的官網上提交的，用的是 Northwest 提供的地址，沒有遇到什麼問題。

記得一定要提交喔!不然會被罰錢，而且罰得不輕呢!

## 美國公司要怎麼報稅?

報稅可能是最讓人頭痛的部分了。不過別擔心，我找到了一位很專業的會計師 Viswa Koll，他專門處理非美國居民的稅務問題。

基本上，如果你跟我一樣是賣數位產品，主要需要注意這幾點:

1. 每年 1/15 前要提交 1120/5472 表單
2. 5 月前要提交年度報表
3. 如果你的收入都不是來自美國，基本上是不用繳美國稅的

但是稅務問題真的很複雜，我強烈建議大家找專業的會計師諮詢，不要自己亂來喔!

## 台灣這邊要怎麼報稅?

這個問題我也問了很多人，得到的答案都不太一樣。不過根據財政部的規定，如果你的基本所得額未達 670 萬元，基本上是不用額外繳稅的。

但是，如果你在台灣有正式的公司或工作室，有開發票給客戶的話，那就還是要按照台灣的規定來報稅喔!

這部分真的很複雜，我建議大家還是找個專業的會計師諮詢比較保險。畢竟稅務問題可大可小，千萬不要掉以輕心!
